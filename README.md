## Requirements
Ensure you have the following installed on your system

## Git

```
pacman -S git
```
## Stow

```
pacman -S stow
```
## Installation
Clone the repo using git

```
$ git clone git@github.com/TechDev-01/dotfiles-hypr
$ cd dotfiles
```
Use GNU stow to generate the symlinks

```
$ stow .
```
